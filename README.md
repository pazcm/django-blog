A simple blog app written using Django.

### Blog features:
- User updatable
- Admin page

### Technologies used:

- [Django](https://www.djangoproject.com/weblog/2015/jun/25/roadmap/)
- HTML Templates
- CSS
- Git
- [Travis](https://travis-ci.org/)
- WhiteNoise library
- Heroku
